## Recent Projects
### 🔍 Self-RAG: Learning to Retrieve, Generate, and Critique Through Self-Reflection

A deep learning research project extending the Self-RAG framework 
(Asai et al., 2023) at small scale, comparing three retrieval-augmented generation 
pipelines on open-domain question answering.

### What I Built
- **Vanilla RAG** — always-retrieve baseline using FAISS vector search over 60k 
  Wikipedia passages
- **Prompted Self-RAG** — prompt-based approximation of Self-RAG's reflection 
  token mechanism using Llama 3.1 8B and requiring no fine-tuning

### Tech Stack
`Python` `FAISS` `Sentence-Transformers` `HuggingFace Transformers` `Llama 3.1 8B` 
`Ollama` `PopQA Dataset` `Matplotlib`

### Paper Reference

Asai et al. (2023). *Self-RAG: Learning to Retrieve, Generate, and Critique through 
Self-Reflection.* arXiv:2310.11511
